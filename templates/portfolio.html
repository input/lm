Ludoly

ludoly.co

Role: Full-stack development

Stack: Python, Django, PostgreSQL, NGINX, Django REST Framework (DRF), Django Template Language (DTL), htmx, Alpine, JavaScript, CSS

Ludoly offers a free, daily collection of puzzles inspired by classic games such as sudoku and minesweeper. Features include: a custom user model, user accounts and profiles, a DRF-based API, puzzle generation, subscription-based access to an archive of past puzzles, and subscription management, billing, and payment integration via Stripe.
